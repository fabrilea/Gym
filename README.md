# Gym App

Aplicación fullstack para administración de un gimnasio.

## Descripción general

Este proyecto centraliza la operación diaria de un gimnasio, incluyendo la gestión de socios, registro de asistencias, facturación mensual, importaciones/exportaciones y trazabilidad de acciones administrativas.

## Arquitectura

- Backend API desarrollado con NestJS.
- Frontend web desarrollado con React + Vite.
- Persistencia en PostgreSQL con migraciones versionadas.

## Módulos funcionales

- Autenticación y autorización por roles.
- Gestión de socios.
- Control de check-ins.
- Facturación y estado mensual.
- Importación y exportación de datos.
- Auditoría de eventos críticos.

## Objetivo del sistema

Proveer una base sólida para la operación interna, con reglas de negocio centralizadas, seguridad por capas y una estructura modular para facilitar mantenimiento y evolución.
